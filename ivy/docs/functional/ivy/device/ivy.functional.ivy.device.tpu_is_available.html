tpu_is_available#

ivy.tpu_is_available()[source]#

Determine whether a TPU is available to use, with the backend framework.

Return type:

bool

Returns:

ret – Boolean, as to whether a tpu is available.

Examples

>>> print(ivy.tpu_is_available())
False
