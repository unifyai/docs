---
title: 'Get Dataset Entry'
api: 'GET /v0/datasetv2/{name}/entry/{id}'
---
Retrieve a specific entry from a dataset.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="name" type="string" required="true">
Dataset name (can include forward slashes)
</ParamField>

<ParamField query="id" type="string" required="true">
Entry ID
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://api.unify.ai/v0/datasetv2/my_dataset/entry/123' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/datasetv2/my_dataset/entry/123"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("GET", url, headers=headers)

print(response.text)
```

</RequestExample>
