---
title: 'Delete Dataset'
api: 'DELETE /v0/datasetv2/{name}'
---
Delete a dataset and all its corresponding entries.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="name" type="string" required="true">
Dataset name (can include forward slashes)
</ParamField>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url 'https://api.unify.ai/v0/datasetv2/my_dataset' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/datasetv2/my_dataset"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("DELETE", url, headers=headers)

print(response.text)
```

</RequestExample>
