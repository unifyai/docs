---
title: "Create or update a secret"
api: 'POST /v0/assistant/{assistant_id}/secret'
---
Creates a new secret for an assistant or updates an existing one. Secrets are used to store API keys and tokens for external services.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="assistant_id" type="integer" required="true">

</ParamField>

#### Body

<ParamField body="secret_name" type="string" required="true">
Unique name/key for the secret (e.g., 'openai_api_key', 'github_token').
</ParamField>

<ParamField body="secret_value" type="string" required="true">
The secret value (API key, token, etc.).
</ParamField>

<ParamField body="description" type="string | null" >
Optional description of what this secret is used for.
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.unify.ai/v0/assistant/<assistant_id>/secret' \
  --header "Authorization: Bearer $UNIFY_KEY"
  --header 'Content-Type: application/json' \
  --data '{
    "secret_name": "openai_api_key",
    "secret_value": "sk-...",
    "description": "OpenAI API key for GPT-4 access"
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/assistant/<assistant_id>/secret"

headers = {"Authorization": "Bearer <token>"}

json_input = {"secret_name": "openai_api_key", "secret_value": "sk-...", "description": "OpenAI API key for GPT-4 access"}

response = requests.request("POST", url, json=json_input, headers=headers)

print(response.text)
```
