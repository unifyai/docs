---
title: "Update secret"
api: 'PUT /v0/assistant/{assistant_id}/secret/{secret_name}'
---
Updates an existing secret's value and/or description.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="assistant_id" type="integer" required="true">

</ParamField>

<ParamField query="secret_name" type="string" required="true">

</ParamField>

#### Body

<ParamField body="secret_value" type="string | null" >
New secret value.
</ParamField>

<ParamField body="description" type="string | null" >
New description for the secret.
</ParamField>

<RequestExample>

```bash cURL
curl --request PUT \
  --url 'https://api.unify.ai/v0/assistant/<assistant_id>/secret/<secret_name>' \
  --header "Authorization: Bearer $UNIFY_KEY"
  --header 'Content-Type: application/json' \
  --data '{
    "secret_value": "sk-new-...",
    "description": "Updated OpenAI API key"
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/assistant/<assistant_id>/secret/<secret_name>"

headers = {"Authorization": "Bearer <token>"}

json_input = {"secret_value": "sk-new-...", "description": "Updated OpenAI API key"}

response = requests.request("PUT", url, json=json_input, headers=headers)

print(response.text)
```
