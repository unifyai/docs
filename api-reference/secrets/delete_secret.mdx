title: "Delete secret"
api: 'DELETE /v0/assistant/{assistant_id}/secret/{secret_name}'
---
Deletes a specific secret from an assistant.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="assistant_id" type="integer" required="true">

</ParamField>

<ParamField query="secret_name" type="string" required="true">

</ParamField>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url 'https://api.unify.ai/v0/assistant/<assistant_id>/secret/<secret_name>' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/assistant/<assistant_id>/secret/<secret_name>"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("DELETE", url, headers=headers)

print(response.text)
```
