---
title: "Create Plot"
api: 'POST /v0/logs/plot'
---
Create a new shareable plot.

Supports two modes:
1. Direct config: Provide explicit plot_config
2. Description-based: Provide natural language description for LLM inference

Requires project:read permission on the target project.
LLM inference (if used) is billed to the caller's account.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Body

<ParamField body="plot_config" type="any | null" >
Direct plot configuration
</ParamField>

<ParamField body="description" type="string | null" >
Natural language description for LLM inference
</ParamField>

<ParamField body="project_config" type="object" required="true">
Project and logs configuration
</ParamField>

<ParamField body="title" type="string | null" >
Optional title for the plot
</ParamField>

<RequestExample>

curl --request POST \

  --header "Authorization: Bearer $UNIFY_KEY" \
