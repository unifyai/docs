---
title: "Update Plot"
api: 'PATCH /v0/logs/plots/{token}'
---
Update a plot.

Requires project:write permission on the plot's project.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="token" type="string" required="true">

</ParamField>

#### Body

<ParamField body="title" type="string | null" >
New title for the plot
</ParamField>

<ParamField body="plot_config" type="any | null" >
New plot configuration
</ParamField>

<ParamField body="project_config" type="any | null" >
New project configuration
</ParamField>

<RequestExample>

```bash cURL
curl --request PATCH \
  --url 'https://api.unify.ai/v0/logs/plots/<token>' \
  --header "Authorization: Bearer $UNIFY_KEY"
  --header 'Content-Type: application/json' \
  --data '{}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/logs/plots/<token>"

headers = {"Authorization": "Bearer <token>"}

json_input = {}

response = requests.request("PATCH", url, json=json_input, headers=headers)

print(response.text)
```
