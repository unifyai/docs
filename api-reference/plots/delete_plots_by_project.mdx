---
title: "Delete Plots By Project"
api: 'DELETE /v0/logs/plots'
---
Delete all plots for a project, optionally filtered by context.

Requires project:write permission on the target project.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Body

<ParamField body="project_name" type="string" required="true">
Name of the project
</ParamField>

<ParamField body="context" type="string | null" >
Optional context to filter by (deletes all if not specified)
</ParamField>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url 'https://api.unify.ai/v0/logs/plots' \
  --header "Authorization: Bearer $UNIFY_KEY" \
  --header 'Content-Type: application/json' \
  --data '{}'
```

```python Python

url = "https://api.unify.ai/v0/logs/plots"
```
