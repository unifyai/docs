---
title: "Send a message to an assistant and get a response"
api: 'POST /v0/assistant/message'
---
Sends a message to a specified assistant and waits for the next reply from that assistant.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Body

<ParamField body="assistant_id" type="integer" required="true">
The ID of the assistant to message.
</ParamField>

<ParamField body="contact_id" type="integer" required="true">
The ID of the contact sending the message. Currently only '1' (the user) is supported.
</ParamField>

<ParamField body="message" type="string" required="true">
The message content.
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.unify.ai/v0/assistant/message' \
  --header "Authorization: Bearer $UNIFY_KEY" \
  --header 'Content-Type: application/json' \
  --data '{
    "contact_id": 1,
    "message": "Hello!"
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/assistant/message"

headers = {"Authorization": "Bearer <token>"}

json_input = {"contact_id": 1, "message": "Hello!"}

response = requests.request("POST", url, json=json_input, headers=headers)

print(response.text)
```
