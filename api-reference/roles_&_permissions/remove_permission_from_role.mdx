---
title: "Remove Permission From Role"
api: 'DELETE /v0/organizations/{organization_id}/roles/{role_id}/permissions/{permission_id}'
---
Remove a permission from a custom role.

Only organization owners can modify role permissions.
Cannot modify system roles.

:param request_fastapi: FastAPI request object.
:param organization_id: Organization ID.
:param role_id: Role ID.
:param permission_id: Permission ID to remove.
:param session: Database session.
:return: Updated role.
