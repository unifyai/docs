---
title: "List Organization Roles"
api: 'GET /v0/organizations/{organization_id}/roles'
---
List all roles available to an organization (system roles + custom roles).

Only organization members can view roles.

:param request_fastapi: FastAPI request object.
:param organization_id: Organization ID.
:param session: Database session.
:return: List of roles.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="organization_id" type="integer" required="true">

</ParamField>
