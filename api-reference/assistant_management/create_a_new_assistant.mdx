---
title: "Create a new assistant"
api: 'POST /v0/assistant'
---
Creates a new assistant for the authenticated user with the specified configuration. This action will deduct credits from the user account.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Body

<ParamField body="first_name" type="string | null" >
First name of the assistant
</ParamField>

<ParamField body="surname" type="string | null" >
Surname of the assistant
</ParamField>

<ParamField body="age" type="integer | null" >
Age of the assistant
</ParamField>

<ParamField body="weekly_limit" type="number | null" >
Weekly time limit for the assistant in hours
</ParamField>

<ParamField body="max_parallel" type="integer | null" >
Maximum number of parallel tasks the assistant can handle
</ParamField>

<ParamField body="nationality" type="string | null" >
Assistant's nationality
</ParamField>

<ParamField body="profile_photo" type="string | null" >
URL to the assistant's profile photo
</ParamField>

<ParamField body="profile_video" type="string | null" >
URL to the assistant's profile video
</ParamField>

<ParamField body="desktop_url" type="string | null" >
URL to the assistant's desktop profile/page
</ParamField>

<ParamField body="user_local_desktop" type="string | null" >
User's local desktop operating system
</ParamField>

<ParamField body="about" type="string | null" >
Brief description about the assistant
</ParamField>

<ParamField body="email" type="string | null" >
Email of the assistant
</ParamField>

<ParamField body="voice_id" type="string | null" >
Id of the provider voice to use for the assistant
</ParamField>

<ParamField body="voice_provider" type="string | null" >
Provider of the selected voice (e.g., 'elevenlabs', 'openai')
</ParamField>

<ParamField body="voice_mode" type="string | null" >
The type of voice interaction, either text-to-speech (tts) or speech-to-speech (sts).
</ParamField>

<ParamField body="user_phone" type="string | null" >
Contact phone number of the user
</ParamField>

<ParamField body="user_whatsapp_number" type="string | null" >
WhatsApp number of the user
</ParamField>

<ParamField body="create_infra" type="boolean | null"  default="true">
Whether to create the infrastructure for the assistant
</ParamField>

<ParamField body="phone" type="string | null" >
Phone number of the assistant (just for testing purposes)
</ParamField>

<ParamField body="phone_country" type="string | null"  default="US">
Country code for phone number provisioning (e.g., US, GB)
</ParamField>

<ParamField body="pre_hire_chat" type="array | null" >
A list of chat messages from the pre-hire conversation to be logged.
</ParamField>

<ParamField body="timezone" type="string | null" >
Timezone of the assistant in IANA format
</ParamField>
