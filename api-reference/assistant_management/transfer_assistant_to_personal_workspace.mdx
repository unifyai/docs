---
title: "Transfer assistant to personal workspace"
api: 'POST /v0/assistant/{assistant_id}/transfer/to-personal'
---
Transfers an organizational assistant to the user's personal workspace.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="assistant_id" type="integer" required="true">

</ParamField>

#### Body

<ParamField body="delete_logs" type="boolean"  default="true">
Whether to delete related logs from the org 'Assistants' project.
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.unify.ai/v0/assistant/<assistant_id>/transfer/to-personal' \
  --header "Authorization: Bearer $UNIFY_KEY"
  --header 'Content-Type: application/json' \
  --data '{}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/assistant/<assistant_id>/transfer/to-personal"
```
