---
title: "Transfer assistant to organization"
api: 'POST /v0/assistant/{assistant_id}/transfer/to-org'
---
Transfers a personal assistant to an organizational workspace.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="assistant_id" type="integer" required="true">

</ParamField>

#### Body

<ParamField body="organization_id" type="integer" required="true">
Target organization ID to transfer the assistant to.
</ParamField>

<ParamField body="transfer_logs" type="boolean"  default="true">
Whether to transfer existing logs from personal 'Assistants' project to org 'Assistants' project.
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.unify.ai/v0/assistant/<assistant_id>/transfer/to-org' \
  --header "Authorization: Bearer $UNIFY_KEY"
  --header 'Content-Type: application/json' \
  --data '{
    "organization_id": 123
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/assistant/<assistant_id>/transfer/to-org"

headers = {"Authorization": "Bearer <token>"}

json_input = {"organization_id": 123}

response = requests.request("POST", url, json=json_input, headers=headers)

print(response.text)
```
