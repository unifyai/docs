Convert rotation axis unit vector \(\mathbf{e} = [e_x, e_y, e_z]\) and rotation angle \(θ\) to quaternion \(\mathbf{q} = [q_i, q_j, q_k, q_r]\).

[reference]

axis_angle – Axis to rotate about and angle to rotate [batch_shape,4]

Returns:

ivy_mech.orientation.quaternion.euler_to_quaternion(euler_angles, convention='zyx', batch_shape=None)[source]#
Convert \(zyx\) Euler angles \(\mathbf{θ}_{abc} = [ϕ_a, ϕ_b, ϕ_c]\) to

quaternion \(\mathbf{q} = [q_i, q_j, q_k, q_r]\).

`[reference]

<https://en.wikipedia.org/wiki/Conversion_between_quaternions_and_Euler_angles>`_

euler_angles

Euler angles, in zyx rotation order form [batch_shape,3]

convention

The axes for euler rotation, in order of L.H.S. matrix multiplication. (Default value = ‘zyx’)

batch_shape

Shape of batch. Inferred from inputs if None. (Default value = None)

ret

Quaternion [batch_shape,4]

Generate random quaternion \(\mathbf{q} = [q_i, q_j, q_k, q_r]\), adhering to maximum absolute rotation angle.

[reference]

Parameters:

  • max_rot_ang – Absolute value of maximum rotation angle for quaternion. Default value of \(π\).

  • batch_shape – Shape of batch. Shape of [1] is assumed if None. (Default value = None)

Returns:

ret – Random quaternion [batch_shape,4]
